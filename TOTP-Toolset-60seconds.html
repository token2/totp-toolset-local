Time Drift OTP Code

How It Works

This calculator generates Time-based One-Time Passwords (TOTP) for different time offsets. The algorithm follows the RFC 6238 standard.

Time Drifts: Shows OTP values for -120, -60, 0, +60, and +120 seconds from the current time.

Base32 Seed: The secret key used to generate OTP codes. Keep this secure!
